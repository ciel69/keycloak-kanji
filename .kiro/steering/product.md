# Product Overview

This is a customized Keycloak authentication and identity management deployment with Russian localization and Telegram identity provider integration.

## Purpose

Provides authentication services for the Kanji Flow application, enabling users to authenticate through standard methods and Telegram.

## Key Features

- Russian-localized Keycloak interface
- Telegram identity provider support for social login
- PostgreSQL-backed user and session management
- Production-ready configuration with HTTPS proxy support
- Containerized deployment for easy scaling and maintenance

## Deployment Target

The service is deployed at `keycloak.kanji-flow.online` and serves as the authentication backend for the Kanji Flow ecosystem.
